# Screenshots

This folder contains app screenshots for the README.

## How to Take Screenshots

### Using Xcode Simulator

1. **Run the app** in Xcode simulator (⌘+R)
2. **Navigate to the screen** you want to capture
3. **Take screenshot** using one of these methods:
   - Press **⌘+S** in the simulator window
   - Click **File → New Screen Shot** in simulator menu
   - Use **⌘+Shift+5** (macOS screenshot tool)

4. **Screenshots are saved to Desktop** by default
5. **Move them to this folder** and rename appropriately

### Recommended Screenshots

Capture these screens for the README:

1. **auth.png** - Login/signup screen
2. **map.png** - Main map view with location
3. **exploration.png** - Map showing exploration progress (future feature)
4. **profile.png** - User profile/settings (future feature)

### Best Practices

- Use **iPhone 14 Pro simulator** for consistency (1179 x 2556 resolution)
- Capture in **light mode** for better visibility
- Ensure **status bar looks clean** (full battery, good signal)
- Use **real-looking data** (not test@test.com)
- **Portrait orientation** for mobile screenshots

### Optional: Screenshot Framing

For a more professional look, you can:
1. Use tools like [screenshots.pro](https://screenshots.pro)
2. Add device frames using [mockuphone.com](https://mockuphone.com)
3. Create a simple collage in Preview or Figma
